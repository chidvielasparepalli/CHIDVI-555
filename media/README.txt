Drop your custom wake media in this folder.

Default names:
- interface_video.mp4
- background_song.mp3

The desktop wake phrase is:
- wake up Jarvis

Other supported automatic names:
- jarvis_interface.mp4
- wake_interface.mp4
- hud.mp4
- interface.mp4
- jarvis_background.mp3
- wake_song.mp3
- theme.mp3
- song.mp3

For custom file names, create config/media.json using config/media.json.example.
